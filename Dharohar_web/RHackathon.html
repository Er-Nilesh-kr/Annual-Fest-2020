Rules

Rules and Regulations:-

• QUH (Quantum University Hackathon) 2020 Organised by App-Club is a six-hour realtime based problem solving hackathon event aimed solving some of the most Pressing Problems of the Time.
• The responsibility of the components and the materials required for developing the prototype during the Hackathon lies solely on the participating teams.
• Travelling & Lodging must be borne by the participants.
• Except basic facility (internet/electricity/working space) nothing will be provided so it is advised that the participants come prepared.
• The entire idea need not be fully implemented however, the submission should be functional so that it can be reviewed by the judges.
• The developer/developers of the solution will have all rights and own the IP of the product. However, all code needs to be in public domain (open source) so that it can be evaluated by the judges.
• All team members must be from the same institute.
• Team members can be from different academic years.
• Once team member name and details are registered. It could not be edited or changed. 5 Team members can be from any appropriate stream of technical background e.g. Btech/BSc/BCA/Mtech/MSc/MCA etc.
• A team can select 1 problem statemnets and propose solutions for the same. Though they can only work on this during the event duration.
• There is no restriction on the number of teams from one college.
